* **Modular:** Tambah atau hapus fitur lewat sistem plugin/command dengan mudah.
* **Ringan:** Membutuhkan sumber daya server yang minimal.
* **Multi-Fungsi:** Cocok untuk bot moderasi, bot bisnis, hingga bot hiburan berbasis AI.
